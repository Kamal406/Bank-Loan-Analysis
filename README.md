# 🏦 Bank Loan Analysis Dashboard

## 📌 Project Overview

This project analyzes bank loan applications to identify the key factors affecting loan approval. Python was used for data cleaning and exploratory data analysis (EDA), while Power BI was used to build an interactive dashboard for business insights.

---

## 📊 Dataset

The dataset contains applicant information, including:

- Applicant Income
- Coapplicant Income
- Loan Amount
- Loan Amount Term
- Credit History
- Education
- Property Area
- Loan Status

---

## 🛠 Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- Excel

---

## 📈 Dashboard KPIs

- Total Loan Applications
- Approved Loans
- Rejected Loans
- Approval Rate
- Average Loan Amount
- Average Applicant Income

## 💡 Key Insights

- Credit History was the strongest factor affecting loan approval.
- Applicants with a positive credit history had a significantly higher approval rate.
- Higher total income increased the likelihood of loan approval.
- Larger loan amounts were associated with lower approval rates.

---

## ✅ Business Recommendations

- Encourage applicants to maintain a strong credit history.
- Review rejected applications with high income and good financial profiles.
- Improve financial awareness programs for first-time applicants.
